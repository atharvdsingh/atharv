# Who is Atharv?

Welcome to my blog! I am **Atharv Dev Singh**, a passionate Full Stack Developer who is deeply in love with creating dynamic, intuitive, and modern web applications. 

## My Journey

My journey into software development started with a simple curiosity about how things work on the internet. Over time, this curiosity evolved into a full-blown passion for coding. I enjoy working across the entire stack, from designing pixel-perfect, responsive user interfaces on the frontend to architecting robust, scalable databases and APIs on the backend.

## What I Do

I specialize in building premium and minimalistic web applications. I believe that good design is just as important as clean code. My tech stack typically revolves around modern JavaScript frameworks like React and Next.js, combined with powerful backend solutions.

## Beyond the Code

When I'm not writing code or debugging, I'm usually exploring new technologies, experimenting with design ideas, or writing about my experiences in the tech world. This blog is a place for me to share my learnings, thoughts, and projects with the community.
